# ポモドーロタイマー

ポモドーロ・テクニックを実践するための、ブラウザ上で動作するシンプルなタイマーアプリケーションです。

サーバーやインストールは不要で、`index.html` をブラウザで開くだけで利用できます。

HTML5・CSS3・JavaScript のみで実装されてています。

---

## ポモドーロ・テクニックとは

ポモドーロ・テクニックは、作業と休憩を一定のリズムで繰り返すことで集中力を維持する時間管理手法です。

一般的な流れは次のとおりです。

1. 一定時間（本アプリではデフォルト25分）集中して作業する
2. 短い休憩（デフォルト5分）を取る
3. 作業と休憩を繰り返す
4. 一定回数（デフォルト4回）作業を完了したら、長めの休憩（デフォルト15分）を取る

---

## 主な機能

### タイマー操作

| 機能 | 説明 |
|---|---|
| 開始 | カウントダウンを開始する |
| 一時停止 | カウントダウンを止め、残り時間を保持する |
| リセット | タイマーを停止し、作業モードの初期時間に戻す。セッション数もリセットする |

### 自動モード切替

- **作業終了時** → 短休憩モードへ自動切替（設定した回数ごとに長休憩モードへ切替）
- **休憩終了時** → 作業モードへ自動切替し、次のフェーズを自動開始

### ステータス表示

画面上部のモード表示は、現在の状態に応じて次のように切り替わります。

| 状態 | 表示 |
|---|---|
| 開始前 | 停止中 |
| 一時停止中 | 一時停止 |
| 作業中 | 作業中 |
| 短休憩中 | 休憩中 |
| 長休憩中 | 長休憩中 |

### セッションカウント

作業時間を1回完了するたびに「今日の完了回数」が1増加します。日付が変わるとカウントは自動的にリセットされます。

### 通知

時間終了時に以下が行われます。

- アラーム音の再生（Web Audio API によるビープ音）
- 画面へのメッセージ表示
  - 作業終了時：「作業終了です。休憩を開始します。」
  - 休憩終了時：「休憩終了です。次の作業を開始しましょう。」

### 設定変更

「設定」ボタンから各種時間を変更できます。設定はブラウザの `localStorage` に保存され、次回起動時も保持されます。

| 項目 | 初期値 | 設定範囲 |
|---|---|---|
| 作業時間 | 25分 | 1～120分 |
| 短休憩時間 | 5分 | 1～60分 |
| 長休憩時間 | 15分 | 1～120分 |
| 長休憩までの回数 | 4回 | 1～10回 |

### 画面デザイン

モードに応じて背景色が変わり、現在の状態を視覚的に把握できます。

| モード | 色 |
|---|---|
| 作業 | 赤系 |
| 短休憩 | 緑系 |
| 長休憩 | 青系 |

PC・スマートフォンの両方で快適に操作できるレスポンシブデザインです。

## 使用方法

### 1. アプリの起動

プロジェクトルートの `index.html` をブラウザで開きます。

```
pomodoro/index.html
```

[こちら](https://glbyymt.github.io/pomodorotimer/)からでも起動できます。



対応ブラウザ：Google Chrome、Microsoft Edge、Firefox

### 2. タイマーの開始

1. メイン画面で「**開始**」ボタンを押す
2. 残り時間が `MM:SS` 形式でカウントダウンされる
3. 作業時間が終了すると、自動的に休憩モードへ切り替わる

### 3. 一時停止・再開

- 作業中に「**一時停止**」を押すと、カウントダウンが止まる（ステータスは「一時停止」に変わる）
- 再度「**開始**」を押すと、残り時間から再開する

### 4. リセット

「**リセット**」を押すと、タイマーが停止し、作業モードの初期時間に戻ります。今日の完了回数も0にリセットされます。

### 5. 設定の変更

1. 「**設定**」ボタンを押して設定画面を開く
2. 各項目の数値を入力する
3. 「**保存**」で設定を反映する（「**キャンセル**」で変更を破棄してメイン画面に戻る）

入力値が範囲外の場合はエラーメッセージが表示されます。

## 技術仕様

| 項目 | 内容 |
|---|---|
| 言語 | HTML5、CSS3、JavaScript（フレームワーク不使用） |
| データ保存 | ブラウザ `localStorage` |
| 音声 | Web Audio API（外部ファイル不要） |
| サーバー | 不要（静的ファイルとして動作） |

### localStorage に保存されるデータ

- 作業時間・短休憩時間・長休憩時間・長休憩までの回数
- 今日の完了回数（日付情報と合わせて保存）
