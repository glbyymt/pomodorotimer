# ポモドーロタイマー 要求仕様書

## 1. 概要

ポモドーロ・テクニックを実践するためのシンプルなタイマーアプリケーションを作成する。

本アプリはブラウザ上で動作し、サーバーは使用しない。
HTML5、CSS3、JavaScriptのみで実装する。

---

## 2. 目的

利用者が以下を簡単に実施できることを目的とする。

- 作業時間の計測
- 休憩時間の計測
- 作業と休憩の繰り返し
- 集中時間の可視化

---

## 3. 動作環境

### 対応ブラウザ

- Google Chrome
- Microsoft Edge
- Firefox

### 動作形態

- ローカルHTMLとして実行可能
- インターネット接続不要

---

## 4. 画面仕様

### メイン画面

表示項目

|項目|説明|
|---|---|
|モード表示|作業中 / 休憩中|
|残り時間|MM:SS形式|
|開始ボタン|タイマー開始|
|一時停止ボタン|タイマー停止|
|リセットボタン|初期状態へ戻す|
|設定ボタン|各種時間設定|
|セッション数|完了した作業回数|

---

## 5. 機能要件

### 5.1 タイマー開始機能

利用者が開始ボタンを押下するとタイマーを開始する。

#### 初期値

- 作業時間：25分
- 短休憩：5分
- 長休憩：15分

---

### 5.2 カウントダウン機能

タイマー実行中は1秒ごとに残り時間を更新する。

表示形式

```
25:00
24:59
24:58
```

---

### 5.3 一時停止機能

利用者が一時停止ボタンを押下した場合

- カウントダウンを停止する
- 残り時間は保持する

---

### 5.4 リセット機能

利用者がリセットボタンを押下した場合

- タイマーを停止する
- 作業時間初期値に戻す
- セッション状態を初期化する

---

### 5.5 モード切替機能

作業時間終了時

```
作業モード
↓
短休憩モード
```

へ自動切替する。

---

### 5.6 長休憩機能

作業セッションを4回完了した場合

```
作業
↓
長休憩
```

へ切り替える。

---

### 5.7 通知機能

時間終了時に以下を行う。

- アラーム音再生
- 画面メッセージ表示

例

```
作業終了です。
5分間休憩しましょう。
```

---

### 5.8 セッションカウント機能

作業時間を完了するたびにカウントを増加する。

例

```
今日の完了回数：3
```

---

### 5.9 設定変更機能

利用者は以下を変更できる。

|項目|範囲|
|---|---|
|作業時間|1～120分|
|短休憩|1～60分|
|長休憩|1～120分|
|長休憩までの回数|1～10回|

---

## 6. 非機能要件

### レスポンス

- ボタン操作後100ms以内に反応すること

### データ保存

ブラウザのlocalStorageを使用する。

保存対象

- 作業時間
- 短休憩時間
- 長休憩時間
- セッション数

---

## 7. UI要件

### デザイン方針

- シンプル
- 視認性重視
- PC・スマホ両対応

### 色分け

|モード|色|
|---|---|
|作業|赤系|
|短休憩|緑系|
|長休憩|青系|

---

## 8. エラー処理

### 不正入力

以下の場合はエラーメッセージを表示する。

- 数字以外
- 許容範囲外

例

```
1～120の範囲で入力してください
```
